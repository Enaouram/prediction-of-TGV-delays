# prediction-of-TGV-delays
Ce projet vise à développer un modèle de prédiction du retard des Trains à Grande Vitesse (TGV) en utilisant des techniques de machine learning et d'analyse de données. Notre objectif est d'améliorer la fiabilité des prévisions de retard des TGV en exploitant diverses données historiques. Chaque banche du Repo traite une ou des approches spécifiques, merci pour votre intérêt !
